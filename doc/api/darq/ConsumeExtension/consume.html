void consume ()

Fully consumes this iterable.

This method will fully consume the iterable, visiting every element once but performing no action. Nothing gets returned from this method.

Implementation

void consume() {
  checkNullError(this);
  final iterator = this.iterator;
  while (iterator.moveNext()) {}
}
