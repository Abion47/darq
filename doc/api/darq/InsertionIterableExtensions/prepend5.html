prepend5 method

Iterable<T> prepend5 (T v1 T v2 T v3 T v4 T v5)

Inserts five elements at the beginning of the iterable.

Takes the specified elements and inserts them at the beginning of the iterable.

Example:

void main() {
  final list = [5, 6, 7];
  final result = list.prepend5(0, 1, 2, 3, 4);

  // Result: [0, 1, 2, 3, 4, 5, 6, 7]
}

Implementation

Iterable<T> prepend5(T v1, T v2, T v3, T v4, T v5) sync* {
  yield v1;
  yield v2;
  yield v3;
  yield v4;
  yield v5;
  yield* this;
}
