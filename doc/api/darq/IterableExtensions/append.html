append method

Iterable<T> append (T value)

Inserts an element to the end of the iterable.

Takes the specified element and inserts it at the end of the iterable.

Example:

void main() {
  final list = [0, 1, 2, 3];
  final result = list.append(4);

  // Result: [0, 1, 2, 3, 4]
}

Implementation

Iterable<T> append(T value) sync* {
  yield* this;
  yield value;
}
