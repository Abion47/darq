firstOrDefault method

T firstOrDefault ({T defaultValue })

Returns the first element in the iterable.

If the iterable is empty, the value specified by defaultValue will be returned instead. If defaultValue is omitted, the returned value will be null.

Example:

void main() {
  final list = <int>[];
  final result = list.firstOrDefault(defaultValue: -1);

  // Result: -1
}

This method is an iterable-consuming method. All lazy iterables will be iterated fully in the process of calculating the result.
