Inserts three elements at the beginning of the iterable.

Takes the specified elements and inserts them at the beginning of the iterable.

Example:

void main() {
  final list = [3, 4, 5];
  final result = list.prepend3(0, 1, 2);

  // Result: [0, 1, 2, 3, 4, 5]
}

Implementation

Iterable<T> prepend3(T v1, T v2, T v3) sync* {
  checkNullError(this);
  yield v1;
  yield v2;
  yield v3;
  yield* this;
}
